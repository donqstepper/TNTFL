post /game/add/json
(gameAddJsonPost)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/x-www-form-urlencoded

Form parameters

redPlayer (required)
Form Parameter — The player on red
redScore (required)
Form Parameter — The red team score format: int32
bluePlayer (required)
Form Parameter — The player on blue
blueScore (required)
Form Parameter — The blue team score format: int32

Return type

Game

Example data

Content-Type: application/json
{
  "date" : 123456789,
  "red" : "",
  "blue" : {
    "score" : 123,
    "achievements" : [ {
      "name" : "aeiou",
      "description" : "aeiou"
    } ],
    "rankChange" : 123,
    "newRank" : 123,
    "name" : "aeiou",
    "href" : "aeiou",
    "skillChange" : 1.3579000000000001069366817318950779736042022705078125
  },
  "positionSwap" : true
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses
